Installation
------------

To install h5hep, clone the source repository from Github and install
with setup.py.

On the command line, enter:

.. code:: bash

    $ git clone git@github.com:mattbellis/h5hep.git
    $ cd h5hep
    $ python setup.py install

You may need to have root (sudo) permissions for the last step, depending
on your installation.
